ACTIVIDAD 1: Elementos de Páginas Web

I. Orígenes de la Web

La web ha evolucionado desde sus inicios, pasando por varias etapas, cada una marcada por avances significativos en la interacción y los servicios ofrecidos.

1. Web 2.0

La Web 2.0, conocida como la "web social", se caracteriza por el uso de tecnologías interactivas que permiten la creación de contenido generado por el usuario, redes sociales, blogs y plataformas colaborativas.

2. Web 3.0

La Web 3.0 es una evolución de la web centrada en la inteligencia artificial, la descentralización, la integración de datos y la personalización a través de la web semántica.

3. La W3C y los estándares

La W3C (World Wide Web Consortium) es una organización que define los estándares y pautas para el desarrollo de tecnologías web. Su objetivo es promover la interoperabilidad y el desarrollo abierto de la web.

II. Lenguajes de etiquetas

Los lenguajes de etiquetas son esenciales para la estructuración de contenido web. A continuación se detallan los principales lenguajes utilizados.

1. SGML

SGML (Standard Generalized Markup Language) es un estándar que define la estructura general para documentos de marcado, el cual sirvió de base para la creación de HTML.

2. XML

XML (Extensible Markup Language) es un lenguaje de marcado utilizado para almacenar y transportar datos. A diferencia de HTML, XML se centra en la definición de estructuras de datos y no en la presentación.

3. HTML

HTML (Hypertext Markup Language) es el lenguaje estándar para la creación de páginas web. Utiliza etiquetas para estructurar el contenido de las páginas de manera jerárquica.

4. XHTML

XHTML (Extensible Hypertext Markup Language) es una versión más estricta y modular de HTML que se basa en XML para mejorar la interoperabilidad y la accesibilidad.

5. HTML5

HTML5 es la última versión de HTML y ofrece nuevas características como soporte para multimedia, formularios avanzados, geolocalización, y más, sin necesidad de complementos adicionales.

III. Otros lenguajes relacionados al desarrollo web

Aparte de los lenguajes de marcado, existen varios lenguajes y tecnologías que permiten mejorar la funcionalidad y el diseño de las páginas web.

1. CSS/CSS3

CSS (Cascading Style Sheets) es un lenguaje utilizado para definir el estilo de las páginas web. CSS3 es la última versión de CSS, que incluye nuevas propiedades para diseño responsivo, animaciones, y transiciones.

2. JavaScript

JavaScript es un lenguaje de programación utilizado para agregar interactividad a las páginas web, como validaciones de formularios, animaciones y la manipulación dinámica de contenido.

3. Ajax

AJAX (Asynchronous JavaScript and XML) es una técnica que permite actualizar partes de una página web sin necesidad de recargar la página completa, mejorando la experiencia del usuario.

4. Tecnologías y lenguajes del lado del servidor

Los lenguajes del lado del servidor, como PHP, Python, y Ruby, permiten generar contenido dinámico en el servidor antes de enviarlo al cliente, mejorando la funcionalidad de la web.

5. Navegadores

Los navegadores web, como Chrome, Firefox, Safari y Edge, son aplicaciones que permiten visualizar y navegar por páginas web.

6. Motores renderizados

Los motores de renderizado, como Blink (usado en Chrome) y Gecko (usado en Firefox), son los encargados de procesar y mostrar el contenido HTML y CSS en la pantalla del usuario.

IV. Herramientas de desarrollo

Existen herramientas que facilitan el diseño, desarrollo y la colaboración en proyectos web.

1. Software WYSIWYG

Los editores WYSIWYG (What You See Is What You Get) permiten crear páginas web de manera visual, sin necesidad de escribir código. Ejemplos populares incluyen Adobe Dreamweaver y Microsoft FrontPage.

2. Otras alternativas

Existen otras herramientas de diseño que facilitan la creación de interfaces y prototipos, entre ellas:

  • Figma: Herramienta de diseño colaborativo basada en la web que permite crear interfaces de usuario de manera interactiva.
  • InVision: Plataforma de diseño para crear prototipos interactivos y colaborar en equipos de trabajo.
  • Zeplin: Herramienta que ayuda a diseñadores y desarrolladores a colaborar y entregar especificaciones precisas de diseño.
  • GitHub: Plataforma de control de versiones y colaboración para proyectos de software.

3. Inspección de código desde el navegador

Los navegadores modernos permiten inspeccionar el código HTML y CSS de cualquier página web mediante herramientas de desarrollo integradas. Estas herramientas ayudan a depurar, modificar y entender el código de una página web en tiempo real.
